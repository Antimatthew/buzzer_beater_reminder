# NBA压哨绝杀提醒程序

这是一个Python程序，可以监控NBA比赛，并在比赛最后2分钟且分差小于5分时发送桌面通知，帮助NBA球迷不错过可能的压哨绝杀时刻。

## 功能特点

- 自动从虎扑官网（https://nba.hupu.com/games）获取NBA实时比分和比赛状态
- 监控第四节或加时赛最后2分钟的比赛
- 当分差小于5分时发送桌面通知
- 避免对同一场比赛重复提醒
- 记录运行日志，便于排查问题

## 安装依赖

1. 确保已安装Python 3.6或更高版本
2. 安装所需的Python包：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 克隆或下载本项目到本地
2. 安装依赖包
3. 运行主程序：

```bash
python nba_game_reminder.py
```

4. 程序将开始监控NBA比赛，并在满足条件时发送桌面通知
5. 按 `Ctrl+C` 可以随时退出程序

## 工作原理

1. 程序每分钟从虎扑NBA实时比分页面抓取数据
2. 分析比赛状态，识别第四节或加时赛的比赛
3. 检查剩余时间是否在2分钟内，且分差是否小于5分
4. 满足条件时，发送桌面通知提醒用户

## 注意事项

- 程序依赖于虎扑网站的页面结构，请在虎扑网站结构发生变化时更新代码中的选择器
- 桌面通知功能可能需要操作系统的相关权限
- 建议在NBA比赛日运行程序，以获得最佳体验

## 日志

程序运行日志保存在 `nba_reminder.log` 文件中，可以通过查看日志了解程序运行状态和可能的错误。

## 自定义配置

如果需要修改提醒条件（如剩余时间或分差阈值），可以直接编辑 `nba_game_reminder.py` 文件中的相关参数。
